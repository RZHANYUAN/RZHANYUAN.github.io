在云服务器上托管html简单网页

RZHANYUAN Lv1
  2025-12-14 17:30:32 2025-12-14 17:30:32 Created   2025-12-14 18:19:17 2025-12-14 18:19:17 Updated  

将从 “准备资源” 到 “绑定域名并公开访问”，帮你整理完整的 HTML 托管。
我将以阿里云云服务器为例。

前期准备

1、拥有阿里云账号，并获取 ECS Windows 实例。

实操 HTML 托管

一、安装 IIS Web 服务器

（1）远程连接

1、找到所要部署的服务器，打开控制台。
2、远程连接 ECS Windows 实例（通过下载 RDP 文件登录）。

（2）开启 IIS Web 服务器功能

1、在 Windows 桌面，按下Win+R打开 “运行”，输入control打开控制面板。
2、选择 “程序和功能” →点击左侧 “启用或关闭 Windows 功能”。
当前页面：

跳过 “开始之前” 页面

1、勾选左下角 “默认情况下跳过此页 (S)”（方便后续操作）；
2、点击右下角 “下一步 (N)”。

选择 “安装类型”

1、进入 “安装类型” 页面：
2、保持默认选中 “基于角色或基于功能的安装”；
3、点击 “下一步 (N)”。

选择 “目标服务器”

1、进入 “目标服务器” 页面：
2、系统会自动选中当前 ECS 服务器（列表里唯一的那个）；
3、点击 “下一步 (N)”。

选择 “服务器角色”（关键：勾选 IIS）

1、进入 “服务器角色” 页面：
2、找到并勾选 “Web 服务器 (IIS)”；
3、弹出 “添加功能” 提示框，直接点 “添加功能”；
4、确认 “Web 服务器 (IIS)” 已勾选后，点击 “下一步 (N)”。

选择 “功能”

1、进入 “功能” 页面：
2、不需要额外勾选，直接点击 “下一步 (N)”。

确认 “Web 服务器 (IIS)”

1、进入 “Web 服务器角色 (IIS)” 页面：
2、直接点击 “下一步 (N)”（看一眼说明即可）。

选择 “IIS 角色服务”

1、进入 “角色服务” 页面：
2、持默认勾选（系统会自动选常用的 Web 功能，足够托管 HTML）；
3、点击 “下一步 (N)”。

确认安装

1、进入 “确认” 页面：
2、检查一下 “Web 服务器 (IIS)” 已被选中；
3、点击 “安装 (I)”，等待系统自动完成安装（过程约 1-2 分钟）。

安装完成

1、安装结束后，点击 “关闭”，回到服务器管理器。

二、上传 HTML 文件到 IIS 目录

找到 IIS 默认网站目录

IIS 的默认网站文件存储路径是：C:\inetpub\wwwroot（可通过 “此电脑”→打开 C 盘→依次进入inetpub→wwwroot文件夹）

上传 HTML 文件到该目录

● 方法 1：直接在远程桌面内操作（最简单）
在本地电脑找到你的 HTML 文件（比如index.html，若有 CSS/JS 等文件需一起上传）；
复制这些文件；
切换到 ECS 的远程桌面窗口，打开C:\inetpub\wwwroot文件夹，右键粘贴文件。

● 方法 2：通过 “文件资源管理器” 远程访问（可选）
在本地电脑打开 “文件资源管理器”，地址栏输入：
\[ECS公网IP]\c$
（比如\123.45.67.89\c$）
输入 ECS 的管理员账号（Administrator）和密码，进入 C 盘；
导航到inetpub\wwwroot文件夹，直接复制粘贴文件。

访问你的 HTML 页面

回到 ECS 实例详情页，复制实例的公网 IP 地址；
打开本地浏览器，在地址栏输入：
http://[ECS公网IP]
（比如格式例http://123.45.67.89）
即可看到你的 HTML 页面。

三、开放 ECS 安全组 80 端口

1、获取 ECS 的公网 IP
回到阿里云控制台的 ECS 实例页面（在 “我的试用”→“管理关联实例” 进入）；
在实例列表中找到你的服务器，复制它的 “公网 IP 地址”（比如120.79.xx.xx）。
2、确认 ECS 的防火墙开放 80 端口
IIS 默认用 80 端口提供网页服务，需要确保 ECS 的防火墙允许外部访问 80 端口：
在 ECS 实例页面，找到 “安全组” 选项（通常在实例详情页的 “网络与安全” 分类下）；
点击安全组名称，进入规则配置页；
检查 “入方向规则” 中是否有 “80 端口（HTTP）允许所有 IP 访问” 的规则：
如果没有，点击 “添加规则”，协议选 “HTTP (80)”，授权对象填0.0.0.0/0（允许所有人访问），保存。
3、测试访问（让别人打开链接）
你自己先在本地浏览器输入：http://[ECS公网IP]（比如http://120.79.xx.xx），确认能看到你的 HTML 页面；
把这个链接发给别人，对方直接在浏览器打开这个地址，就能访问到你的页面了。

四、配置域名解析

登录域名注册商平台→进入域名 “解析管理”；
添加 2 条 A 记录：
主机记录@，记录值填 ECS 公网 IP；
主机记录www，记录值填 ECS 公网 IP；
保存解析（生效需 5-30 分钟）。

五、在 IIS 中绑定域名

在 ECS 远程桌面，运行inetmgr打开 IIS 管理器；
左侧展开服务器→点击 “网站”→选中 “默认网站”；
右侧操作栏点击 “绑定”→“添加”：
类型：HTTP；
端口：80；
主机名：填写你的域名（如yourdomain.com）；
点击 “确定” 完成绑定。

六、测试访问

本地浏览器输入http://你的域名（如http://yourdomain.com）；
确认页面正常显示，他人即可通过该域名访问你的 HTML 网站。
